Desafio do Módulo de HTML

Entendendo o Desafio

Para concluir o desafio proposto devemos utilizar algumas tags que foram apresentadas em aula.

Objetivos

O principal objetivo é conseguir utilizar algumas tags em uma estrutura HTML básica, abaixo segue uma lista com algumas das tags de estrutura apresentadas,
porém algumas já foram previamente utilizadas para a estruturação básica deste desafio, como por exemplo as tags <h1, h2, h3, h4, h5, h6>,
assim como também a tag <p> que compõe os parágrafos desta página.

Lista das tags utilizadas
  1. <h1> até <h6>
  2. <mark> e <small>
  3. <i>, <u> e <strong>
  4. <a>, <hr>, <sub> e <sup>
  5. <blockquote>

Exemplos

  • <h1> até <h6>

    • Com estas tags podemoscriar parágrafos, mas não apenas isso, elas são utilizadas para dar importâncias aos blocos mais importantes do site.

  • <mark> e <small>

    • Mark é bastante utilizado para dar conotação à frases e palavras específicas, como um marcador de texto faria, já o small tansforma o texto em uma versão ligeiramente menor.

  • <i>, <u> e <strong>

    • Bastente utilizados para editar o texto em parágrafos, o <i> deixa as palavras em itálico, enquanto o <u> sublinha as palavras, e por último <strong> faz com que as palavras fiquem em NEGRITO.

  • <a>, <hr>, <sub> e <sup>

    • As tags <a>, <hr>, <sub> e <sup> são bastante utilizads para edição tanto das palavras quanto da estrutura do HTML, enquanto a tag <a> adiciona um hiperlink ao texto, podendo levar o usuário à outras páginas ou outra seção da página atual como por exemplo Clique Aqui! as tags <sub> e <sup> criam textos subscritos e sobrescritos e finalizando com o <hr> que adiciona uma quebra de linha capaz de separar os conteúdos da página.

  • <blockquote>

    • Muito utilizado para fazer citações, aqui em baixo pode ver como essa tag se difere de um texto normal escrito com a tag <p>.
